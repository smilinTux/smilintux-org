How do I reverse a string in Python?

You can reverse a string in Python using slicing:

s = "hello"
reversed_s = s[::-1]
print(reversed_s)  # "olleh"

Or using the reversed() built-in with join():

reversed_s = "".join(reversed(s))

Which approach is faster?

The slicing approach s[::-1] is generally faster because it is implemented in C under the hood and avoids the overhead of a Python iterator.

A quick benchmark:

import timeit
t1 = timeit.timeit('s[::-1]', setup='s="hello world"', number=1_000_000)
t2 = timeit.timeit('"".join(reversed(s))', setup='s="hello world"', number=1_000_000)
print(f"slice: {t1:.3f}s   reversed: {t2:.3f}s")
