## Developer Notes for Reviewer

This extension:
- Does NOT collect or transmit user data to any third-party server
- Does NOT inject ads, track browsing, or monitor non-AI-platform pages
- Host permissions are limited to the 5 supported AI platforms and localhost (optional local agent)
- The `localhost` host permission is optional â€” used only when the user configures a local SKComm agent endpoint in Options; the extension works fully offline without it
- All network calls are either to user-configured localhost endpoints or blocked entirely
- Content scripts run only on the listed AI platform URLs (not all_urls)
- `scripting` permission is used only to inject a user-approved context prompt into the AI input field on command â€” not for arbitrary script injection

Single purpose: export and restore AI conversation context.
